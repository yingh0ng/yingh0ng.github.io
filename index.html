使用-Djava.ext.dirs时的问题

Posted on 2018-08-13 | In java

用cmd运行java程序时报了一个“javax.net.ssl.SSLException: java.lang.RuntimeException: Could not generate DH keypair”。
根据百度上几个说法，包括加jar包，换jdk之类的都不行。发现生成那个keypair会用到\jre\lib\ext里面的jar包，而我的cmd中的命令
有 -Djava.ext.dirs这个参数。原来这个参数会把系统原来的$JAVA_HOME\jre\lib\ext覆盖掉，故在我的”-Djava.ext.dirs=”后面加上
$JAVA_HOME/jre/lib/ext(这个路径要写在其他路径前面？？)。问题到这就解决了。

记一个使用PageHelper做分页时的特殊情况

Posted on 2018-07-17 | In java

在使用PageHelper4.1.6做分页的时候，有两点需注意。

1.当”select count(0)”的total小于size的时候，PageHelper不做分页（无论page传的是多少，
最后组装的sql后面limit部分的offset都是0）。

2.当传的offset>(total-1)时，最后PageHelper都会把offset改为total-size-1。即当传的page
大于最后一页时，永远的都是返回最后一页的数据。
